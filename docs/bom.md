## Notes

- All Adafruit breakout boards use Qwiic/STEMMA QT connectors for easy I2C wiring
- The STEMMA QT cables can daisy-chain all I2C devices together
- Make sure to get USB-C version of TP4056 (not micro-USB)
- 85A TPU filament recommended for flexible bracelet joints
